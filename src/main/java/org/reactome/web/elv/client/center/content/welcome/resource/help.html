Welcome to the Reactome Pathway Browser, a tool for visualizing and interacting with pathways.

To view a Pathway Diagram either:

  1. Click on a pathway name in the hierarchical list on the left
  2. Return to the Homepage (button top left) and text search using a pathway, protein or compound name, or an accession number (e.g. Q9HCN6). In the resulting list, click on pathway name to open it in the Pathway Browser.

To see a key to pathway diagrams click the Diagram Key link top right

For a detailed explanation see Section 3 of the Reactome User Guide

For help please contact us (help@reactome.org)

The Pathway Browser includes tools for several types of analysis, detailed in Section 5 of the Reactome User Guide.
